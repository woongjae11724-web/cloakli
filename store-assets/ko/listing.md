## 이름

Cloakli

## 짧은 설명 (132자 이내)

화면 공유 전에 웹페이지에서 민감한 정보를 간편하게 가립니다.

## 상세 설명

회의, 강의, 녹화 중에 화면을 공유하시나요? 이메일 제목, 계좌 잔액, 주문 내역, 개인 맞춤 추천… 웹에는 보여주고 싶지 않은 것이 가득합니다. 탭을 닫고, 녹화를 나중에 잘라내는 건 번거롭고 잊기 쉽습니다.

Cloakli는 누군가 보기 전에, 그 부분을 미리 가립니다.

사용 방법

1. Cloakli 아이콘을 누르고 "가릴 영역 선택"을 클릭합니다.
2. 화면이 고정되어 hover 효과나 팝업이 방해하지 않습니다.
3. 가리고 싶은 요소를 클릭하고 범위를 고릅니다: 이 요소만 / 현재 페이지 유형 / 사이트 전체.
4. 끝 — 깔끔한 가림이 덮이고, 다음에 방문해도 자동으로 다시 적용됩니다.

주요 기능

- 화면 고정 선택 모드로 클릭 한 번에 요소 가리기
- 사이트 재방문 시 저장된 가림 자동 재적용
- 적용 범위 선택: 이 요소만 / 현재 페이지 유형 / 사이트 전체
- 클릭 한 번으로 사이트 가림 일시중지 — 저장된 가림은 유지
- 단축키로 페이지를 잠시 보이게 했다가 다시 가리기
- 옵션 페이지에서 모든 저장된 가림 관리

Free와 Pro

- Free: 1개 사이트에서 최대 3개 가림 저장 ('이 요소만' 범위)
- Pro: 가림·사이트 무제한, 페이지 유형·사이트 전체 범위, 최대 3대 기기

개인정보 보호

- 가림 규칙은 브라우저 안에 로컬로만 저장됩니다
- 계정 불필요, 분석·추적 없음
- Free 사용자는 네트워크 요청이 전혀 발생하지 않습니다
- Pro 라이선스 확인 시에도 해시된 키 식별자만 전송 — 페이지 내용, 방문 기록, 개인정보는 절대 전송하지 않습니다
- 전체 정책: https://cloakli.pages.dev/privacy/

알아두세요

Cloakli는 화면에 보이는 내용을 시각적으로 덮을 뿐, 원본 데이터를 삭제하지 않습니다. Chrome 내부 페이지(chrome://, 웹 스토어)에서는 동작할 수 없고, 사이트 구조가 바뀌면 가림을 다시 만들어야 할 수 있습니다. "화면 공유"를 누르기 전에 가림 상태를 꼭 직접 확인하세요.
