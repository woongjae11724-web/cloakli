# Chrome Web Store Listing — English

## Name

Cloakli

## Short description (132 chars max)

Hide sensitive information on webpages before sharing your screen.

## Detailed description

Sharing your screen in a meeting, a lecture, or a recording? Email subjects, account balances, order history, personal recommendations — the web is full of things you'd rather not show. Closing tabs and cropping recordings afterwards is tedious and easy to forget.

Cloakli lets you hide those parts before anyone sees them.

HOW IT WORKS

1. Click the Cloakli icon and choose "Select an area to hide".
2. The page freezes so hover effects and popups don't get in your way.
3. Click the element you want to hide and pick a scope: this element only, this page type, or the whole site.
4. Done — a clean mask covers it, and it comes back automatically the next time you visit.

FEATURES

- One-click element hiding with a frozen-page selection mode
- Saved masks re-apply automatically when you revisit a site
- Scopes: this element only / this page type / entire site
- Pause all masks on a site with one click — your masks stay saved
- Temporarily reveal a page with a keyboard shortcut, then restore
- Manage every saved mask from the options page

FREE AND PRO

- Free: up to 3 saved masks on 1 site (single-element scope)
- Pro: unlimited masks and sites, page-type and site-wide scopes, up to 3 devices

PRIVACY

- Your mask rules are stored locally in your browser
- No account required; no analytics or tracking
- Free users make no network requests at all
- Pro license checks send only a hashed key identifier — never page content, browsing history, or personal data
- Full policy: https://cloakli.pages.dev/privacy/

GOOD TO KNOW

Cloakli draws a visual cover over content on your screen — it does not delete the underlying data, and it can't run on Chrome internal pages (chrome://, the Web Store). If a site redesigns, a saved mask may need to be re-created. Always double-check your masks before you hit "Share screen".
