# 스토어 스크린샷 촬영 가이드 (1280×800, 5장)

촬영 대상: `store-assets/demo/index.html` (허구 데이터 픽스처).
실행: 저장소 루트에서 `npx.cmd serve store-assets/demo` → http://localhost:3000 접속.
브라우저 창 내부 크기를 1280×800으로 맞춘다 (DevTools → Ctrl+Shift+M → Responsive 1280×800 후 캡처해도 됨).

## 촬영 규칙

- 데모 픽스처 외의 실제 웹사이트를 배경으로 쓰지 않는다 (브랜드/실데이터 금지).
- 개인 프로필, 실제 북마크 바, 다른 확장 아이콘이 프레임에 들어가지 않게 한다 (게스트 프로필 권장).
- 언어: 영어 스크린샷을 기본으로 등록하고, 한국어 listing에는 동일 장면의 한국어 UI 캡처를 써도 된다 (Chrome 언어를 한국어로 바꾸면 확장 UI가 한국어로 표시됨).
- 프로모 타일(선택): small 440×280, marquee 1400×560 — `og-image-placeholder.svg`를 기반으로 제작 예정, 아직 없음.
