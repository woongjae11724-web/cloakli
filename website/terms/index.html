1. 서비스

Cloakli는 화면 공유 전에 웹페이지의 일부를 시각적으로 가릴 수 있는 Chrome 확장 프로그램입니다. 계정 없이 사용할 수 있으며, 선택형 유료 플랜("Cloakli Pro")은 Lemon Squeezy에서 구매한 라이선스 키로 활성화됩니다.

2. 사용권(라이선스)

  • Free 기능은 개인·업무 용도로 무료로 사용할 수 있습니다.
  • Pro 라이선스는 최대 3대의 활성 기기에서 Pro 기능을 사용할 수 있는 개인적·비독점적·양도 불가한 사용권입니다.
  • 라이선스 키의 공유·재판매·재라이선스·공개 게시는 금지됩니다. 오·남용되거나 광범위하게 공유된 키는 비활성화될 수 있습니다.
  • 가림 결과를 사실과 다르게 표현하는 용도(가림 = 데이터 삭제인 것처럼)로 사용해서는 안 됩니다(4조 참고).
  • 관련 법률이 허용하는 범위 내에서, 라이선스 검증 서비스를 역설계하거나 검증을 우회하려는 시도는 금지됩니다.

3. Free와 Pro

Free는 1개 사이트에서 '이 요소만' 범위로 최대 3개의 가림을 저장할 수 있습니다. Pro는 이 제한이 없으며 페이지 유형·사이트 전체 범위를 추가로 제공합니다. 기능 구성은 개선 과정에서 달라질 수 있으며, 유료 기능의 중대한 축소는 이 사이트에서 사전 안내합니다.

4. 가림의 한계와 사용자의 확인 책임

  • Cloakli는 화면에 표시되는 내용을 시각적으로 덮을 뿐, 웹사이트의 원본 데이터를 삭제하거나 변경하지 않습니다.
  • 웹사이트 구조가 바뀌면 저장된 가림이 다시 적용되지 않을 수 있으며, 특정 웹사이트와의 호환을 보장하지 않습니다.
  • 화면 공유·녹화를 시작하기 전에 가림이 의도대로 표시되는지 반드시 직접 확인해야 합니다. 공유하는 내용에 대한 책임은 사용자에게 있습니다.
  • Cloakli는 보안 제품이 아니며, 다른 소프트웨어의 페이지 데이터 접근을 차단하지 않고, 비밀번호 관리자나 DLP 시스템을 대체하지 않습니다.
