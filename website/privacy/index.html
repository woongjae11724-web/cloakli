Summary

Cloakli draws visual masks over parts of web pages before you share your screen. Your masks and settings live in your browser. Cloakli has no accounts and no analytics. The only network communication the extension makes is with our license server, and only for verifying a Pro license.

1. Data stored locally in your browser

The following is stored with Chrome's local extension storage on your device and is not sent anywhere by itself:

  • Mask rules: the CSS selector used to find the element again, the site hostname, the scope (element / page type / site), an optional page pattern (URL path), a structural fingerprint (tag/class/position and hashed link identifiers — never the text itself), and a random rule ID
  • Per-site pause state
  • Onboarding completion state
  • A random installation ID (generated with a random UUID; it contains no personal information)
  • If you activate Pro: a license session token, the last 4 characters of your license key (for display), and a cached license status

The actual text, names, emails, images, or any other content of the pages you mask is not stored.

This website itself stores one value in your browser's localStorage: your language preference ("en" or "ko"). No cookies, no analytics.

2. Data sent to our license server

Only if you activate or verify a Pro license, the extension contacts our license server (a Cloudflare Worker at cloakli-license.mycloakli.workers.dev) and sends:

  • Your license key (once, during activation — the server stores only a SHA-256 hash of it)
  • The random installation ID
  • The license session token (for re-verification)
  • The extension version

The server keeps request timestamps and limited technical records used for abuse prevention (rate limiting) and error handling. As with any internet service, your IP address is processed by the network infrastructure (Cloudflare) that delivers the request; we do not use it to identify you.

3. Data that is never sent

  • Web page text or HTML
  • Email contents, video titles, channel names
  • Passwords or form input
  • Screenshots or the content of images
  • The actual text you masked
  • Your browsing history

Cloakli contains no analytics, telemetry, or advertising trackers, and the release build is automatically checked to ensure the only outbound network call is to the license server.

4. Third-party services

ServiceRole
Chrome Web Store (Google)Distributes and updates the extension. Google's own policies apply to the store.
CloudflareHosts the license server (Workers/D1) and this website (Pages).
Lemon SqueezyMerchant of record. Handles checkout, payment, invoices, license key issuing, and subscription management. Your payment card details are entered on and processed by Lemon Squeezy — Cloakli's servers never receive or store card details.

5. Data retention

  • Local data: delete individual masks in the extension's "Manage saved masks" page, or remove all local data by uninstalling the extension.
  • License records on the server store only hashes (license key hash, session token hash, installation ID hash) — never the originals.
  • Retention targets: deactivated device records and webhook processing metadata are kept up to 90 days; rate-limit records up to 30 days; active license records are kept while the license is in use and for the period needed for support and legal obligations.

6. Your choices

  • Delete any or all saved masks at any time in the extension
  • Deactivate your license on a device ("Deactivate on this device")
  • Cancel your subscription via Lemon Squeezy (see the refund policy)
  • Contact us for privacy questions or deletion requests: cloakli.support@gmail.com

7. Children

Cloakli is a general-purpose productivity tool and is not directed at children under 13. We do not knowingly collect personal information from children.

8. Changes

If this policy changes, we will update the "Last updated" date above. Material changes will be noted on this page.

요약

Cloakli는 화면 공유 전에 웹페이지 일부를 시각적으로 가리는 도구입니다. 가림 규칙과 설정은 사용자의 브라우저 안에 저장됩니다. 계정 기능과 분석(analytics) 기능이 없으며, 확장 프로그램이 하는 유일한 네트워크 통신은 Pro 라이선스 확인을 위한 라이선스 서버 호출뿐입니다.

1. 브라우저에 로컬로 저장되는 정보

다음 정보는 Chrome 확장 프로그램 로컬 저장소에 사용자의 기기에만 저장되며, 그 자체로는 어디에도 전송되지 않습니다:

  • 가림 규칙: 요소를 다시 찾기 위한 CSS 선택자, 사이트 hostname, 적용 범위(요소/페이지 유형/사이트), 페이지 패턴(URL 경로), 구조 fingerprint(태그/클래스/위치와 해시된 링크 식별자 — 실제 텍스트는 포함하지 않음), 무작위 규칙 ID
  • 사이트별 일시중지 상태
  • 첫 사용 안내(onboarding) 완료 상태
  • 무작위 설치 ID (무작위 UUID로 생성되며 개인정보를 포함하지 않음)
  • Pro 활성화 시: 라이선스 세션 토큰, 표시용 라이선스 키 마지막 4자리, 라이선스 상태 캐시

가린 부분의 실제 텍스트, 이름, 이메일, 이미지 등 페이지 내용 자체는 저장하지 않습니다.

이 홈페이지는 브라우저 localStorage에 언어 선택("en"/"ko") 한 가지만 저장합니다. 쿠키와 분석 도구는 사용하지 않습니다.

2. 라이선스 서버로 전송될 수 있는 정보

Pro 라이선스를 활성화하거나 재확인할 때에만, 확장 프로그램이 라이선스 서버(Cloudflare Worker, cloakli-license.mycloakli.workers.dev)에 다음을 전송합니다:

  • 라이선스 키(활성화 시 1회 — 서버에는 SHA-256 해시만 저장됩니다)
  • 무작위 설치 ID
  • 라이선스 세션 토큰(재확인용)
  • 확장 프로그램 버전

서버는 요청 시각과 오·남용 방지(요청 빈도 제한)·오류 처리에 필요한 제한된 기술 기록을 보관합니다. 다른 인터넷 서비스와 마찬가지로, 요청을 전달하는 네트워크 인프라(Cloudflare)에서 IP 주소가 처리될 수 있으나 이를 사용자 식별에 사용하지 않습니다.

3. 절대 전송하지 않는 정보

  • 웹페이지 본문·HTML
  • 이메일 내용, 영상 제목, 채널명
  • 비밀번호, 입력 폼 내용
  • 화면 캡처, 이미지 내용
  • 사용자가 가린 실제 텍스트
  • 브라우징 기록

Cloakli에는 분석·텔레메트리·광고 추적 코드가 없으며, 출시 빌드는 "라이선스 서버 외의 외부 통신 없음"을 자동 검사로 확인한 뒤에만 만들어집니다.

4. 제3자 서비스

서비스역할
Chrome 웹 스토어 (Google)확장 프로그램의 배포와 업데이트를 담당합니다. 스토어 자체에는 Google의 정책이 적용됩니다.
Cloudflare라이선스 서버(Workers/D1)와 이 홈페이지(Pages)를 호스팅합니다.
Lemon Squeezy공식 판매자(Merchant of Record)로서 결제, 영수증, 라이선스 키 발급, 구독 관리를 처리합니다. 카드 정보는 Lemon Squeezy에서 입력·처리되며 Cloakli 서버는 카드 정보를 받지도, 저장하지도 않습니다.

5. 보관 기간

  • 로컬 데이터: 확장 프로그램의 "저장된 가림 관리"에서 개별 삭제하거나, 확장 프로그램을 제거하면 모두 삭제됩니다.
  • 서버의 라이선스 기록은 해시만 저장합니다(라이선스 키 해시, 세션 토큰 해시, 설치 ID 해시) — 원문은 저장하지 않습니다.
  • 보관 기준: 비활성화된 기기 기록과 웹훅 처리 메타데이터는 최대 90일, 요청 빈도 제한 기록은 최대 30일을 기준으로 하며, 활성 라이선스 기록은 사용 기간과 지원·법적 의무에 필요한 기간 동안 보관합니다.

6. 사용자의 권리

  • 저장된 가림을 언제든 개별/전체 삭제
  • 기기에서 라이선스 비활성화("이 기기에서 비활성화")
  • Lemon Squeezy를 통한 구독 취소 (환불 정책 참고)
  • 개인정보 문의·삭제 요청: cloakli.support@gmail.com
