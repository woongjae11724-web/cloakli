Getting started

Install

Install Cloakli from the Chrome Web Store (see the download page), then pin the icon from the puzzle-piece menu in your toolbar.

Hide an element

  1. Open a regular website and click the Cloakli icon.
  2. Click "Select an area to hide" — the page freezes so hover effects won't get in your way.
  3. Click the part you want to hide, then choose how broadly the mask applies (this element only / this page type / this site).

Delete saved masks

Popup → "Manage saved masks" → delete a single mask, all masks for a site, or everything.

Pause a site

Popup → "Pause masks on this site". Saved masks stay saved and can be resumed anytime.

Plans and licensing

Free limits

Free saves up to 3 masks on 1 site, with single-element scope. When you hit a limit, the extension explains it inline — nothing is silently blocked.

Activate Pro

After purchase you receive a license key by email. Popup → "Enter license key" → paste the key → "Activate Pro".

Re-check or move a license

Popup → "Re-check license" refreshes the status. "Deactivate on this device" frees one of your 3 device slots so you can activate elsewhere.

Billing questions

Payments and invoices are handled by Lemon Squeezy. For cancellations and refunds see the Refund Policy.

Known limitations

  • Chrome internal pages (chrome:// pages, the Chrome Web Store, the new tab page) do not allow extensions to run.
  • Sites that change their structure may need a mask to be re-created.
  • Always verify masks yourself before sharing your screen.

Report a site that doesn't work

Email cloakli.support@gmail.com with:

  • Website domain
  • What you selected
  • Expected behavior
  • Actual behavior
  • Cloakli version (shown in chrome://extensions)
  • Screenshot without sensitive information

Please never include real personal information — no real emails, names, account numbers, or license keys — in screenshots or bug reports.

시작하기

설치

Chrome 웹 스토어에서 Cloakli를 설치한 뒤(다운로드 페이지 참고), 툴바의 퍼즐 아이콘 메뉴에서 Cloakli를 고정하세요.

요소 가리기

  1. 일반 웹사이트에서 Cloakli 아이콘을 클릭합니다.
  2. "가릴 영역 선택"을 누릅니다 — 화면이 고정되어 hover로 내용이 바뀌는 사이트에서도 원하는 부분을 선택할 수 있습니다.
  3. 가릴 부분을 클릭한 뒤 적용 범위(이 요소만 / 현재 페이지 유형 / 사이트 전체)를 선택합니다.

저장된 가림 삭제

팝업 → "저장된 가림 관리"에서 규칙 하나, 사이트 전체, 또는 전부를 삭제할 수 있습니다.

사이트 일시중지

팝업 → "현재 사이트 가림 일시중지". 저장된 가림은 유지되며 언제든 다시 시작할 수 있습니다.

요금제와 라이선스

Free 한도

Free는 1개 사이트에서 '이 요소만' 범위로 최대 3개의 가림을 저장할 수 있습니다. 한도에 도달하면 화면에서 이유를 바로 안내합니다.

Pro 활성화

구매 후 이메일로 라이선스 키를 받습니다. 팝업 → "라이선스 키 입력" → 키 붙여넣기 → "Pro 활성화".

라이선스 다시 확인 / 기기 이동

팝업의 "라이선스 다시 확인"으로 상태를 갱신할 수 있습니다. "이 기기에서 비활성화"를 누르면 3개 기기 슬롯 중 하나가 반납되어 다른 기기에서 활성화할 수 있습니다.

결제 문의

결제와 영수증은 Lemon Squeezy가 처리합니다. 취소·환불은 환불 정책을 확인하세요.

알려진 제한

  • Chrome 내부 페이지(chrome:// 설정, Chrome 웹 스토어, 새 탭)에서는 확장 프로그램이 동작할 수 없습니다.
  • 구조가 바뀐 사이트에서는 가림을 다시 만들어야 할 수 있습니다.
  • 화면 공유 전에는 항상 가림 상태를 직접 확인하세요.
